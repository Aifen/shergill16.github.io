How to Create a PDF file on Microsoft Word

6 steps of how to make a PDF file

Step 1

First,open Microsoft Word

Step 2

Write the document.Use Microsoft Word to write whatever document you'd eventually convert to a PDF.

Step 3

Open file in order to make the document a PDF file

Step 4

Click Print.This is the second option from the bottom of the drop-down menu.

Step 5

Select "PDF." It's the option on the bottom left of the Print menu. Click on the arrow.

Step 6

Select "Save as PDF." This will open a new window that will allow you to save the document.

Now you finally know how to create a PDF file on Microsoft Word
